# Platform Walkthrough

1. Open the app.
2. Check LLM status in the header.
3. Open Agents and create a new agent to verify CRUD.
4. Open Workflow Builder.
5. Run the default business update prompt.
6. Verify that action items appear in the action table.
7. Open WhatsApp Channel and save connector details.
8. Use Send from the WhatsApp chat area to trigger the agent and send final output.
9. Open Monitoring to view persisted logs and message history.
